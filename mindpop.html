升级版的类 Anki 记忆工具

解决"学了就忘"的痛点，通过强制主动输出和科学间隔复习，让记忆更加牢固。

Spring Boot MySQL JWT SM-2算法 AI辅助开发
访问项目 → GitHub源码

01

核心功能

强制主动输出 - "单框连击"交互

  • 创新交互设计：无需手动提交，输入即匹配
  • 防自我欺骗：只有敲对关键词，答案才会砰然跳出
  • 无序答题：任意顺序回答，无需死记硬背顺序

科学间隔复习

基于 Anki SM-2 算法，在遗忘临界点精准推送复习，对抗遗忘曲线。

完整用户系统

JWT 无状态认证、BCrypt 密码加密、严格账户数据隔离。

SM-2 算法核心逻辑

// 新间隔 = 旧间隔 × 简易度系数 × 评级乘数 int newInterval = (int) (oldInterval * (easeFactor / 1000.0) * ratingMultiplier); // 简易度动态调整: 评级好则上升, 差则下降(范围1300-5000) int newEaseFactor = easeFactor + (rating == 4 ? 150 : (rating == 1 ? -250 : 0));

技术架构

后端架构

经典三层架构：Controller → Service → Repository → Entity

  • Spring Boot 2.7.18
  • Spring Data JPA + Hibernate
  • MySQL 8.0
  • JWT (JJWT) + Spring Security
  • Docker + 阿里云部署

数据库核心表

  • users - 用户表（BCrypt加密）
  • quiz - 测验主表
  • answer - 答案表
  • quiz_group - 分组表（多对多）
  • quiz_review_status - SM-2核心数据

评级体系

评级 说明 间隔乘数
重来 完全忘记，需要重新学习 × 间隔衰减
困难 有印象但回忆困难 × 1.2
良好 正确回忆，稍有犹豫 × ease factor
简单 轻松回忆，正确无误 × ease factor × 1.3
